# Content Generator UI (Simple Static Demo)

این یک نسخه ساده و استاتیک از رابط کاربری برای «اتوماسیون تولید مقاله» است. فایل‌های فرانت‌اند اکنون در پوشه `client/` قرار دارند:

- `client/index.html` — صفحه اصلی UI
- `client/assets/styles.css` — استایل ساده و ریسپانسیو (RTL)
- `client/assets/app.js` — منطق فرانت‌اند و شبیه‌سازی تولید بخش‌ها

نحوه اجرا:

1) ساده‌ترین روش: `client/index.html` را مستقیماً در مرورگر باز کنید.

2) یا برای رفتار بهتر (و حل محدودیت‌های CORS)، یک سرور ساده محلی راه‌اندازی کنید. در PowerShell این دستور را اجرا کنید:

```powershell
cd "i:\computer\My projs\other\content-generator-workflow"; python -m http.server 8000
```

سپس به `http://localhost:8000/client/` بروید.

توضیحات مختصر عملکرد:

- وارد کردن پارامترهای اولیه → دکمه «تولید بخش‌ها» → تولید فهرست بخش‌ها (شبیه‌سازی)
- برای هر بخش دکمه «تولید محتوا» را بزنید تا متن نمونه ساخته شود
- دکمه «بررسی SEO» یک گزارش ساده از تراکم کلیدواژه نشان می‌دهد
- پس از تولید همه بخش‌ها، مقاله نهایی نمایش داده می‌شود و می‌توانید آن را دانلود کنید.

این پیاده‌سازی فقط نمایشی است؛ برای اتصال به بک‌اند واقعی کافی است در `assets/app.js` تابع تولید یا فراخوانی API را جایگزین کنید.
